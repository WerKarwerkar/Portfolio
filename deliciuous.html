Delicious vagabond

This project aims to create a dynamic blog site with features such as a 'Latest Posts' carousel for desktop users, responsive design, and a 'Load More' button for additional blog posts. Specific blog pages dynamically generate content based on query strings, and the contact page includes a user-friendly form with JavaScript validation. Developed using JavaScript, HTML, CSS, and leveraging WordPress as a Headless CMS for efficient content management.
